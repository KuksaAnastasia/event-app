# Event App
Mobile application for organizing events.

## Features
- Create events
- Manage participants

## Technology Stack
- Frontend: React Native
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: Firebase Auth
- Notifications: Firebase Cloud Messaging
